# Technical Claim Verifier

## Problem

Manually verifying a technical claim against a large collection of research papers is time-consuming. Relevant evidence may be scattered across different papers and expressed using different terminology. This project aims to retrieve relevant evidence from a bounded research corpus to support technical claim verification.

## v0.1 Scope

- Accept a collection of research papers and a technical claim.
- Search the provided papers for semantically relevant passages.
- Return the top-k candidate passages for further inspection.
